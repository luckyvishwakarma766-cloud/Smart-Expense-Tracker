💸 Spendly — Smart Expense Tracker
A fully functional expense tracking web app in a single HTML file.

🚀 How to Run
Just open index.html in any modern browser. No installation, no server needed.

✨ Features
Dashboard with stats, charts, and recent transactions
Add/delete income & expenses
Budget management with visual progress bars
Analytics: donut chart, monthly comparison, top spending days
AI Financial Advisor powered by Claude API
All data saved in localStorage (persists between sessions)
📁 Files
index.html — The entire app (HTML + CSS + JS in one file)
README.md — This file
🛠 Tech Used
Vanilla HTML, CSS, JavaScript
Chart.js (via CDN) for charts
Tabler Icons (via CDN) for icons
Google Fonts — DM Sans + DM Serif Display
Anthropic Claude API for AI insights
💡 Notes
Demo data is pre-loaded on first launch
Currency is set to ₹ (Indian Rupee) — change ₹ to $ or any symbol to localize
AI Insights tab requires internet (calls Claude API)
